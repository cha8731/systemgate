템플릿 - 버튼

버튼

기본 버튼

  • base_button와 table_button의 따라 크기가 결정되고 bg_white, bg_gray, bg_red, bg_blue, bg_cobalt을 추가하여 다른 색상을 지정할 수 있다.
<button type="button" class="base_button bg_white">base_button</button>
<button type="button" class="base_button bg_gray">bg_gray</button>
<button type="button" class="base_button bg_red">bg_red</button>
<button type="button" class="base_button bg_blue">bg_blue</button>
<button type="button" class="base_button bg_cobalt">bg_cobalt</button>

<button type="button" class="table_button bg_white">table_button</button>
<button type="button" class="table_button bg_white">bg_gray</button>
<button type="button" class="table_button bg_white">bg_red</button>
<button type="button" class="table_button bg_white">bg_blue</button>
<button type="button" class="table_button bg_white">bg_cobalt</button>

테이블 버튼

  • 버튼의 설정 클래스로 base_button를 사용하고 원하는 색상 지정을 위해bg_white 추가한다.
<button type="button" class="tag bg_white">bg_gray</button>
<button type="button" class="tag bg_white">bg_red</button>
<button type="button" class="tag bg_white">bg_blue</button>
<button type="button" class="tag bg_white">bg_cobalt</button>

정렬

  • 신규 사이트 box_left 정렬을 추천하며 기존사이트는 하단 부분에 taL을 사용하면 된다.

[신규]왼쪽 정렬 - box_left {display:flex;}

왼쪽 정렬
<div class="box_left">왼쪽 정렬</div>

[신규]가운데 정렬 - box_center {display:flex;}

가운데 정렬
<div class="box_center">가운데 정렬</div>

[신규]오른쪽 정렬 - box_right {display:flex;}

오른쪽 정렬
<div class="box_right">오른쪽 정렬</div>

[기존]왼쪽 정렬 - taL {text-align:left;}

왼쪽 정렬
<div class="taL">왼쪽 정렬</div>

[기존]가운데 정렬 - taC {text-align:center;}

가운데 정렬
<div class="taC">가운데 정렬</div>

[기존]오른쪽 정렬 - taR {text-align:right;}

오른쪽 정렬
<div class="taR">오른쪽 정렬</div>
