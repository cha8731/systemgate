화면설정

개요

실무에서 사용하는 구성요소와 태그를 웹 접근성을 고려하여 정리하였습니다.
템플릿 화면에서 요소를 사용하기 위해서는 화면설정 부분에 CSS, JS, script 파일을 먼저 적용해주세요.

엘리먼트 적용

<!doctype html>
<html lang="ko">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE-edge" /> <!-- 브라우저 호환성 -->
<meta name="Keywords" content="ex)시스템게이트 사업관리시스템" /> <!-- 검색엔진에 의해 검색되는 단어 지정 -->
<meta name="Description" content="ex) 안녕하세요 시스템게이트 사업관리 시스템입니다." /> <!-- 검색 결과에 표시되는 문자 지정 -->
<meta name="author" content="ex)시스템게이트" /><!--제작자 설정 -->
<meta name="subject" content="ex)시스템게이트 사업관리시스템" /> <!-- 사이트 주요 주제 설정 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 모바일 반응형 페이지 설정 -->
<link href="이미지 경로" rel="shortcut icon" type="image/x-icon" /> <!-- 파비콘 설정 -->
<title>ex)시스템게이트 메인화면</title>
</head>

script

 <!-- 메뉴 -->
<script type="text/javascript">
	var menu=1;
	var sub=1;

	function hide(){
		if(menu && sub){
			$("#gnb .on").next().slideUp("fast");
			$("#gnb .on").removeClass("on");
		}
	}

	$("#gnb > li > a").bind("mouseover focus",function(){
		$("#gnb .on").next().slideUp("fast");
		$("#gnb .on").removeClass("on");

		$(this).addClass("on");
		$(this).next().stop().slideDown("fast");
	})

	$("#gnb").mouseenter(function(){
		menu=0;
	});
	$("#gnb ul").mouseenter(function(){
		sub=0;
	});

	$("#gnb").mouseleave(function(){
		menu=1;
		setTimeout(hide, 500);
	});
	$("#gnb ul").mouseleave(function(){
		sub=1;
		setTimeout(hide, 500);
	});
</script>
