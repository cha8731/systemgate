화면규칙

서체

폰트 종류

1. 나눔고딕 - 문서의 본문에도 잘 쓸 수 있는 고딕 글꼴입니다. 글자 끝의 날카로운 부분을 둥글게 처리해 친근하고 부드러운 느낌입니다.

한글 11,172자, 한자 4,888자, 영문 94자, KS약물 986자

<!-- 웹 폰트 URL -->
<link href="https://hangeul.pstatic.net/hangeul_static/css/nanum-gothic.css" rel="stylesheet">

/* CSS font-family 설정 */
font-family: 'NanumGothicLight';
font-family: 'NanumGothic';
font-family: 'NanumGothicBold';
font-family: 'NanumGothicExtraBold';

2. 본고딕(Noto Sans) - 구글과 어도비 시스템즈가 2014년 7월 출시한 한국어, 중국어 번체와 간체, 일본어와 라틴어, 그리스어, 키릴 자모를 지원하는 한중일 공통 무료 오픈소스 글꼴입니다.

<!-- 웹 폰트 URL -->
@import url(//fonts.googleapis.com/earlyaccess/notosanskr.css);

/* CSS font-family 설정 */
.notosanskr * { 
 font-family: 'Noto Sans KR', sans-serif;
}

3. 프리텐다드 - 본고딕을 기반으로 하여 자간, 행간, 개개별 글자 디자인 등을 조금 더 세련되고 가독성 좋게 다듬은 무료 서체입니다.

/* CSS font-family 설정 */
@font-face {
    font-family: 'Pretendard-Regular';
    src: url('https://cdn.jsdelivr.net/gh/Project-Noonnu/noonfonts_2107@1.1/Pretendard-Regular.woff') format('woff');
    font-weight: 400;
    font-style: normal;
}
