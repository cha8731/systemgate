HTML 구획 제목 요소

  • HTML <h3>–<h5> 요소는 3단계의 구획 제목을 나타냅니다. 구획 단계는 <h3>이 가장 높고 <h5>은 가장 낮습니다.
  • <h1>요소는 로고영역 나타내고, <h2>요소는 서브화면 상단 페이지 타이틀 제목을 나타냅니다.
  <h3>제목 수준 3</h3>
<h4>제목 수준 4</h4>
<h5>제목 수준 5</h5>

버튼

기본 버튼

  • base_button와 table_button의 따라 크기가 결정되고 bg_white, bg_gray, bg_red, bg_blue, bg_cobalt을 추가하여 다른 색상을 지정할 수 있다.
base_button bg_gray bg_red bg_blue bg_cobalt 
<a href="#" class="base_button bg_white">base_button</a>
table_button bg_gray bg_red bg_blue bg_cobalt 
<a href="#" class="table_button bg_white">table_button</a>

<a href="#" class="tag bg_lightGray">tag</a>

	...
}

/* 스크롤바 막대 꾸미기 */
body::-webkit-scrollbar-thumb {
	...
}

/* 스크롤바 트랙 꾸미기 */
body::-webkit-scrollbar-track {
	...
